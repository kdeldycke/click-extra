Bases: ExtraOption

A pre-configured option that is adding a --color/--no-color (aliased by --ansi/--no-ansi) option to keep or strip colors and ANSI codes from CLI output.

This option is eager by default to allow for other eager options (like --version) to be rendered colorless.

Prints the colored version of the CLI.

This is a [copy of the standard @click.version_option() decorator](https://github.com/pallets/click/blob/dc918b48fb9006be683a684b42cc7496ad649b83/src/click/decorators.py#L399-L466) that has been made into a class to allow it to be used with declarative params= argument (fixes [Click #2324 issue](https://github.com/pallets/click/issues/2324)).

For other params see Click’s version_option decorator: https://click.palletsprojects.com/en/8.1.x/api/#click.version_option

Parameters:

  • param_decls (_type_, optional) – _description_, defaults to None

  • version (_type_, optional) – _description_, defaults to None

  • package_name (t.Optional[str], optional) – _description_, defaults to None

  • prog_name (t.Optional[str], optional) – _description_, defaults to None

  • message (str, optional) – _description_, defaults to “%(prog)s, version %(version)s”

  • print_env_info (bool, optional) – _description_, defaults to False

  • version_style (_type_, optional) – adds environment info at the end of the message. Useful to gather user’s details for troubleshooting. Defaults to Style(fg="green").

  • package_name_style (_type_, optional) – style of the version. Defaults to theme.invoked_command.

  • prog_name_style (_type_, optional) – style of the prog_name. Defaults to theme.invoked_command.

  • message_style (_type_, optional) – default style of the message parameter. Defaults to None.

  • env_info_style (_type_, optional) – style of the environment info. Defaults to Style(fg="bright_black").

  • is_flag (bool, optional) – _description_, defaults to True

  • expose_value (bool, optional) – _description_, defaults to False

  • is_eager (bool, optional) – _description_, defaults to True

  • help (str, optional) – _description_, defaults to “Show the version and exit.”

Adds extra-keywords highlighting to Click commands.

This mixin for click.core.Command-like classes intercepts the top-level helper-generation method to initialize the formatter with dynamic settings.

This is implemented here to get access to the global context.
